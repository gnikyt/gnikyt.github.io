Software

Below is a personal list of every-day software I love to use and what keeps me productive.

Ruby

A great programming language for scripts and the web. Completely modular and extendable with a beautiful syntax.

Bitbucket

Easy-to-use Git hosting with great team integration, staff support, and accompanying software.

Brackets

Simplistic, distraction-free code editor with many great extensions to ease development. Great for both front and backend developers.

Vagrant

A great tool to keep your development environment contained. Create, modify, and tear-down without worry.

Rails

Handsdown the most flexible and mature web development framework I've used. Extensive documentation, community, and tools available.

Fedora

With over a decade of Linux, Fedora is my favourite. Child of Red Hat, clean, GNOME-centric distribution for anyone. Even though I'm on OSX these days.

Dash

An awesome, searchable, at-your-finger-tips documenation viewing application for OSX. Docs for languages, tools, and 3rd party contributations.

Flux

Keeps my eyes sane at night. Automatically adjusts screen tint depending on location and the time of day.

Sourcetree

Although I mainly use commandline Git, this is a great GUI tool developed by the guys behind Bitbucket and JIRA. Github & Bitbucket support.

Firefox

An opensource, fast, and extendable browser. I love the built-in tools available to me which aid in development. Their Sync is also a great feature.

Sinatra

When Rails is too big, I turn to Sinatra. A well-documented Ruby framework for small to medium-sized projects.

Noizio

As programmers — we hate distractions. This app for iOS and OSX takes care of that with customizable sounds in an easy interface.

Typora

A beautiful feature-rich Markdown editor which has live editing. Its what I use for all my software docs, personal notes, and my blog posts.

Slack

The best team communication tool out there. A pile of integrations, fun bots, sectioned channels, and more. It helps keep a team organized in real-time.

OneNote

Along with Microsoft's other great tools, OneNote is a great daily hand to keep me organgized in both personal and work capacities.

Wunderlist

Everyone has things to do and this app can help you cross them off. A slick interface, with instant sync support, group sharing, and folder views.

Commandline

Last but not least, the terminal. Who could go without it as a developer?
